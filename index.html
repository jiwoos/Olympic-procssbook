Data Acquisition

We acquired all the datasets that cover 124 years of Olympics. There is one dataset for Tokyo 2020, one for Rio 2016, and one for all the Summer Olympics between 1896 and 2014. As we realized that it wasn't possible to parse the number of medal counts for each country using our current datasets, we also found another dataset (in addition to our earlier three))that has all the medal counts for each summer Olympic year.

We had to parse four datasets differently. We completed parsing 2016 and 2020 in a structure we need it to be. We also have parsed the dataset which covers the other years.

You can find cleaned-up csv files in data folder.
